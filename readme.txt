Place facebook pixel on your website in a simple unobtrusive way

== Description ==

The problem with other facebook pixel plugins is that they are bloated and also add javascript in an obtrusive way.

This plugin keeps it simple, adds the javascript unobtrusively, and makes decisions rather than giving options

== Frequently Asked Questions ==

= How can I check the facebook pixel is installed? =

If you use Chrome browser you can install the Facebook Pixel Helper from their web store.

== Installation ==

1. Upload the entire `lh-simple-facebook-pixel` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings->Facebook Pixel to configure the plugin.
